## Installation

```bash
npx shadcn@latest add avatar
```




Install the following dependencies:

```bash
npm install @radix-ui/react-avatar
```

## Usage

```tsx showLineNumbers
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"
```

