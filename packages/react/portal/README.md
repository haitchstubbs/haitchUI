# @haitch/react-portal

This is a simple wrapper for @floating-ui/dom & @floating-ui/react.
The goal of this package is to provide developer friendly API's & some helper functions for @haitch/ui components requiring portals.
If you are looking to use @floating-ui outside of @haitch/ui, please use the [official release](https://floating-ui.com/docs/getting-started)

## Install
```sh
pnpm add @haitch/react-portal
```
