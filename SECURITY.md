# Security Policy

## Supported Versions

We release security updates for the latest stable version of this project. Please ensure you are using the most recent release.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open an issue**. Instead, email us at haitchstubbs@gmail.com with details and steps to reproduce. We will respond as quickly as possible and coordinate a fix.

## Disclosure Policy

We ask that you give us a reasonable time to address the issue before disclosing it publicly.
